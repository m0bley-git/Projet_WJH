# Weather Judgment Hub

Le projet consiste à développer une application C++/Qt permettant d’évaluer en temps réel la faisabilité d’un décollage d’hélicoptère en fonction des conditions météorologiques. 

L’objectif est d’offrir au pilote ou à l’opérateur au sol une aide à la décision fiable, basée sur des données météo actualisées ou renseignées manuellement.

L’application récupérera les données météorologiques via une API externe ou via une saisie manuelle. 

À partir de ces informations, elle analysera plusieurs paramètres critiques (comme la vitesse du vent, la pression atmosphérique, la visibilité, etc.) et déterminera si le décollage est autorisé ou refusé.
